# Bubble Bobble Godot Clone

Arad Ayntabli - 100845722

## Controls

| Action | Input |
|:-:|:-:|
|Move left | A/left arrow|
|Move right | D/right arrow|
|Jump | W/up arrow|
|Fire | Space|
|Restart | R |

## Implemented Features
- Core movement + world
- Bubble shooting
- Enemies (more on bonus section)
- Trapping
- Popping
- Score
- Lives
- Restart
- Menu and Game over Screen

## Bonus Features
- Made the player an animated sprite for running animation
- Made it so enemies continue to spawn so user can keep playing
- An enemy spawns every 3 seconds in a random location

## Known bugs/limitations
Nothing, except that this github repo probably has less than 5 commits, I didn't know we needed a repo for this assignment

## How to run
Run either Bubble Bobble.exe for Windows or Bubble Bobble.x86_64 for Linux

If running manually, start with menu.tscn
